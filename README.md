# Western States 100 Lottery

Simulation to calculate probability of having an entry ticket selected in the Western States 100 lottery.

All code is in `docs/index.qmd`.

Run `quarto render docs/index.qmd` in terminal to generate the HTML output.

Website: [https://peter-ehmann.github.io/westernStates100-simulation/](https://peter-ehmann.github.io/westernStates100-simulation/)
